## Introdução

Use laços para desenhar uma pista de corrida e criar um jogo de corrida de tartaruga.

### O que você vai fazer

Este projeto introduz laços for através de um divertido jogo de corrida de tartaruga. Laços são usados ​​para desenhar a pista de corrida e fazer as tartarugas moverem um número aleatório de passos a cada ciclo. Se você tem um grupo de pessoas para jogar, cada pessoa escolhe uma tartaruga e a que mais se aproxima é a vencedora.

### O que você vai aprender

Ao fazer o seu jogo de corrida de tartarugas, você aprenderá a:

+ Escrever laços `for` em Python
+ Usar o gerador de números aleatórios em Python
+ Desenhar linhas de cores diferentes com Python Turtle

Este projeto abrange elementos das seguintes vertentes do [Raspberry Pi Currículo de Criação Digital](http://rpf.io/curriculum){:target="_blank"}:
